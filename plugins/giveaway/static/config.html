2 The prize

Tip: write the giveaway text in your DWEEB message, then attach this button to it. Drop these placeholders into the message text and they fill in — and stay live: {prize}, {entries}, {winners}, {winner_count}, {host}, {status}. The winners appear the moment you draw, the entrant count keeps ticking (it also rides the button).
Advanced options

3 Who can enter

By default anyone can enter, once each.
Blocks brand-new alt accounts. Derived from the account's age — no extra setup.

4 Deadline (optional)

Sets a countdown and closes entries when it passes. There's no auto-draw — you draw the winners yourself with the host controls, whenever you're ready. Open them in Discord by clicking the giveaway's button as a host; server managers can also right-click (long-press on mobile) the message → Apps → Message Info → Manage giveaway.

5 When you draw

Use {winners}, {prize}, {entries} and the other placeholders — they're filled in for you.
Server managers can always draw. Add roles here to let others run it too — any host opens the controls by clicking the giveaway's button in Discord.
